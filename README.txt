Para instalar osquery y filebeat en mac simplemente se debe descargar el fichero setup.sh y ejecutarlo de la siguiente manera

sh setup.sh logstash_ip_addr logstash_port username
  siendo logstash_ip_addr y logstash_port la ip y el puerto donde se encuentra la m√°quina de nuestro logstash agent escuchando y username
  el nombre del usuario de mac, para facilitar el acceso a los diferentes directorios
